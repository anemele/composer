# composer 编曲

三种乐谱，分别用于不同场景

一是频率-拍数矩阵谱，用于数值计算，获得音频数据；
二是音名-拍数谱，三是唱名-拍数谱，用于编谱。

频拍谱只由程序生成，人工编写十分麻烦，不推荐。
音名谱灵活性高，可以在十二律框架内任意发挥。
唱名谱基于自然音阶，适用于大部分音乐。

音名谱使用 `C Db D Eb E F Gb G Ab A Bb B` 共十二个音，都用降音符号。
唱名谱用 `1 2 3 4 5 6 7` 共七个音符。
二者都由名-拍有序对组成，有序对的结构为：
`X:r`
拍数相同的有序对可以用逗号隔开，如
`X1,X2,X2:r`
有序对之间用空格、制表符、换行符等分割。
以 `#` 开头的行是注释行，会被程序忽略，可以写歌词等标注。
乐谱文件第一行是相对值，格式为
`[X,R]`
表示 `1=X` 以及所有拍数都要除以 `R`，音名谱无需根音，只写 `[,R]` 即可。
这个 `R` 是一个参考值，推荐使用最小公倍数，以免出现小数；
另外可以用来快速修改整体的拍数。
关于唱名谱，单独数字表示中音，`_` 表示低音，`^` 表示高音。

示例乐谱见 `./sheet` 目录。
